## 라우팅
- 현재 라우터가 제공하는 컴포넌트가 렌더링
```
<router-view> </router-view>
```

- a 태그 처럼 링크를 이동시키는데 차이점은 <br>
  a 태그는 페이지 새로고침이 되고, 라우터 링크는 페이지 리로드하지 않는다.
```
<router-link to="/"> </router-link> 
```
